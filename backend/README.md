# Daily Streaker

Small backend service for tracking daily streaks.

## Quick start

Prerequisites
- Node.js (or your runtime)
- Yarn or npm

Install
```bash
npm install
# or
yarn
```

Run (development)
```bash
npm run dev
# or
node ./src/index.js
```

API docs
- Swagger UI / OpenAPI documentation is served at: `/docs`

Notes
- Adjust start script and entry point to match your project layout.
- Add environment config (PORT, DB connection, etc.) as needed.
